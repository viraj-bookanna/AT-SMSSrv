# AT-SMSSrv
flask srever with AT SMS sending using modem

## Installation

### Clone the repo
```bash
git clone https://github.com/viraj-bookanna/AT-SMSSrv.git
```

### Installing requirements

```bash
cd autoscan-tg-qr
```

```bash
pip install -r requirements.txt
```

## Usage

```bash
python smssrv.py
```
